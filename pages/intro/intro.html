Blockchain Demo

A web-based demonstration of blockchain concepts.

This is a complete rewrite of Anders Brownworth's Blockchain Demo with lots of additional features.
Basically only the idea shown in his excellent Demo Video remains, the code is completely different.

This project is made for only educational purposes to create a dummy blockchain

About this website:
  • Static HTML/JS, so it can be served with GitHub Pages
  • Use AngularJS for rendering the page
  • Expert Mode that can set difficulty
  • Show duration and speed of mining process (in Expert Mode)
  • Toggle between TX/Coinbase and Data view
